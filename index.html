Multi-modal Large Language Models have strong general reasoning ability, but still struggle with spatial understanding in the 3D world. This page presents pySpatial, a visual programming framework that lets MLLMs call spatial tools through Python code generation so they can reconstruct scenes, recover poses, render new views, and reason explicitly in 3D space.

Real-World Application

Robot navigation demo image from the original site.

In the real-world setup, pySpatial reconstructs the scene from limited camera views, infers the camera poses, and then generates a motion plan for a quadrupedal robot to reach the target object. The showcased example contrasts the generated plan with a GPT-4.1 baseline and highlights the geometry-consistent navigation route produced by pySpatial.

Conclusion

pySpatial enhances spatial reasoning by converting image sequences into explorable 3D scenes through zero-shot Python code generation. By composing 3D reconstruction, pose recovery, and novel-view synthesis tools, the system delivers more explicit spatial reasoning and transfers well from benchmark evaluation to physical robot navigation.
